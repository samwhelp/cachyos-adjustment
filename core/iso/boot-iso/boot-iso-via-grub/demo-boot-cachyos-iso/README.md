

# demo-boot-cachyos-iso




## Live Account

| Account  | Value  |
| -------- | ------ |
| Username | `liveuser` |
| Password |  |


run to change password

``` sh
sudo passwd $(whoami)
```


run to remove password

``` sh
sudo passwd -d $(whoami)
```
